What is amsl?

amsl is a an Electronic Resource Management web application that makes use of RDF as data model. This allows the system to adapt to changing modeling requirements by adjusting the used RDF vocabulary. Developed at the Leipzig University Library, amsl has been continuously improved according to requirements as they emerged. Several use-cases have already been implemented in form of extensions. This involves techniques such as rdf-based templating, realtime-search or various file import and processing mechanisms.
